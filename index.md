## 推荐教程

* [Interneting Is Hard](https://www.internetingishard.com/) | Web Development Tutorials For Complete Beginners

  友好型的前端入门教程，教程难度和广度合适，包括 HTML 和 CSS（但不包括框架），网页布局讲得很好。

* [现代 JavaScript 教程](https://zh.javascript.info/)

  超全面细致的 JavaScript 教程，基于**最新的 JavaScript 标准**，难度和广度都超合适新手到进阶级别的，讲解从基础到进阶的知识，当做手册也是很合适的。

* [精通 Bootstrap 4 - 開發超強不解釋](https://www.udemy.com/course/getbootstrap-4/)

  有了 HTML、CSS、JavaScript 基础后可以学习该课程，Bootstrap 4 提供了更多定制化功能，可以快速排出各种网页版型，这个教程将基础和项目结合，超适合练手的。
